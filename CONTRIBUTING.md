# Contributing

## Branch strategy

- `main` — always stable, protected
- `feature/*` — new features
- `fix/*` — bug fixes
- `chore/*` — maintenance (theme updates etc.)

## How to contribute

1. Fork this repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Commit your changes
4. Push and open a Pull Request to `main`
5. Wait for review and approval

## themes.json

`themes.json` is auto-generated. Do not edit it manually.
To update locally: `python3 generate.py`
