# Contributing

## Feature requests & bug reports

Please open an [Issue](https://github.com/eiji1202/ghostty-theme-preview/issues) to request features or report bugs.

## Pull Requests

PRs are welcome, but only the maintainer (@eiji1202) can merge to `main`.

1. Fork this repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Commit your changes
4. Push and open a Pull Request to `main`
5. Wait for review and approval from the maintainer

## themes.json

`themes.json` is auto-generated. Do not edit it manually.
To update locally: `python3 generate.py`
